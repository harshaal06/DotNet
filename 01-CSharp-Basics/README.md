This folder contains small console examples covering fundamental C# concepts. Each subfolder focuses on a specific topic with practical examples.

## Topics Covered

1. **01-Variables-Types** - Variables, data types, type conversion
2. **02-Operators** - Arithmetic, logical, comparison operators
3. **03-Control-Flow** - if/else, switch, loops
4. **04-Arrays-Collections** - Arrays, List, Dictionary, LINQ
5. **05-Methods** - Method definition, parameters, return types
6. **06-Exception-Handling** - try/catch/finally, custom exceptions
7. **07-String-Operations** - String manipulation, formatting
8. **08-DateTime** - Date and time operations

## How to Run

```bash
cd 01-CSharp-Basics/01-Variables-Types
dotnet run
```

## Learning Path

Start with 01-Variables-Types and progress sequentially. Each example builds on previous concepts.
