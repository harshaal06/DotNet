This folder contains examples demonstrating Object-Oriented Programming concepts in C#.

## Topics Covered

1. **01-Classes-Objects** - Basic classes and object creation
2. **02-Encapsulation** - Access modifiers, properties
3. **03-Inheritance** - Base classes, derived classes
4. **04-Polymorphism** - Method overriding, virtual methods
5. **05-Abstraction** - Abstract classes and interfaces
6. **06-Constructors** - Default, parameterized, static constructors
7. **07-Properties-Indexers** - Properties, auto-properties, indexers
8. **08-Access-Modifiers** - public, private, protected, internal

## How to Run

```bash
cd 02-OOPS/01-Classes-Objects
dotnet run
```

## Learning Path

Start with Classes-Objects and progress through each concept. Each example builds on previous knowledge.
